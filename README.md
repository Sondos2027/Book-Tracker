# 📚 مشروع نظام تتبع الكتب (Book Tracker API)

هذا المشروع هو عبارة عن تطبيق ويب (API) متكامل لإدارة سجلات الكتب، تم بناؤه باستخدام إطار العمل **FastAPI** ولغة البرمجيات **Python**.

---

## 💡 فكرة الخدمة
الخدمة هي نظام خلفي (Backend) لإدارة مكتبة رقمية بسيطة. تتيح للمستخدم إمكانية:
* إضافة كتب جديدة إلى قاعدة البيانات.
* عرض قائمة بجميع الكتب المخزنة.
* البحث عن كتاب معين باستخدام معرفه الخاص (ID).
* تعديل بيانات الكتب الموجودة (العنوان، المؤلف، السنة).
* حذف الكتب نهائياً من النظام.

---

## 🐳 كيفية تشغيلها باستخدام Docker
المشروع مجهز بالكامل ليعمل داخل حاوية (Container) لضمان التشغيل السهل على أي جهاز.

1.  **بناء صورة الـ Docker:**
    ```bash
    docker build -t book-tracker-app .
    ```

2.  **تشغيل الحاوية:**
    ```bash
    docker run -d -p 8080:8080 --name my-book-app book-tracker-app
    ```
    *سيصبح النظام متاحاً على الرابط: `http://localhost:8080`*

---

## 🚀 كيفية استخدام الـ API
يوفر النظام واجهة توثيق تفاعلية وسهلة الاستخدام عبر **Swagger UI**.

* **رابط التوثيق:** بعد تشغيل النظام، اذهب إلى `http://localhost:8080/docs`


* **📚  لعرض قائمة الكتب على شكل ملف (JSON):**(http://localhost:8080/books)
  *(تعرض جميع الكتب المخزنة حالياً بصيغة JSON)*.


### العمليات المتاحة (Endpoints):
| العملية | المسار | الوظيفة |
| :--- | :--- | :--- |
| **GET** | `/` | رسالة ترحيبية للتأكد من عمل النظام |
| **GET** | `/books` | عرض قائمة بجميع الكتب |
| **POST** | `/books` | إضافة كتاب جديد (عنوان، مؤلف، سنة) |
| **GET** | `/books/{id}` | عرض تفاصيل كتاب محدد عبر الـ ID |
| **PUT** | `/books/{id}` | تحديث بيانات كتاب موجود |
| **DELETE** | `/books/{id}` | حذف كتاب من النظام |

---

## 🛠️ التقنيات المستخدمة
* **FastAPI**: لبناء الـ API بسرعة وأداء عالٍ.
* **SQLAlchemy**: لإدارة قاعدة البيانات (ORM).
* **SQLite**: قاعدة بيانات خفيفة مدمجة.
* **Docker**: لتغليف وتشغيل التطبيق بسهولة.
