# 📚 نظام إدارة الكتب (Book Tracker API)

هذا المشروع هو المخرج النهائي للمرحلة التأسيسية لبرنامج تطوير الأنظمة. يهدف المشروع إلى بناء خدمة Backend بسيطة تتبع أفضل الممارسات البرمجية.

## 🚀 الهدف من المشروع
- تطبيق مفاهيم **Clean Code** وتنظيم الملفات.
- بناء واجهة برمجية باستخدام **RESTful API**.
- استخدام **GitHub** لإدارة إصدارات الكود و Branches.
- تجهيز الخدمة للتشغيل باستخدام **Docker** .

---

## 🛠️ التقنيات المستخدمة
- **اللغة:** Python 🐍
- **إطار العمل:** FastAPI (لأنه يدعم التوثيق التلقائي و JSON).
- **إدارة البيانات:** Pydantic (للتحقق من صحة المدخلات - Validation).

---

## ⚙️ تعليمات التشغيل
بسبب بعض التعارضات في المنافذ الافتراضية، تم إعداد المشروع ليعمل على المنفذ **8080** لضمان استقرار الخدمة

**لتشغيل هذه الخمة اكتب الامر في Terminal**
   """bash
    uvicorn main:app --reload --port 8080

## 🔗 روابط الوصول السريع

بعد تشغيل الخدمة بنجاح، يمكنك الوصول إلى الروابط التالية لتجربة الـ API:

* **🏠 الصفحة الرئيسية:** [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
  *(تعرض رسالة ترحيبية للتأكد من أن السيرفر يعمل)*.

* **📚 قائمة الكتب (JSON):** [http://127.0.0.1:8080/books](http://127.0.0.1:8080/books)
  *(تعرض جميع الكتب المخزنة حالياً بصيغة JSON)*.

* **🚀 التوثيق التلقائي (Swagger UI):** [http://127.0.0.1:8080/docs](http://127.0.0.1:8080/docs)
  *(أداة تفاعلية رائعة تسمح لك بتجربة الـ API مباشرة دون الحاجة لأدوات خارجية)*.

## 🐳 التشغيل باستخدام Docker
لتشغيل المشروع في بيئة معزولة، استخدم الأوامر التالية:
1. بناء الصورة: `docker build -t book-tracker-app .`
2. تشغيل الحاوية: `docker run -p 8080:8080 book-tracker-app`
